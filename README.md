# isEven()

Returns true if a number is even

## Usage

```ts
import { isEven } from "@crgeary/is-even";

isEven(1); // false
isEven(42); // true
isEven(1337); // false
isEven(2024); // true
isEven(9001); // false
isEven(1000000); // true
```
